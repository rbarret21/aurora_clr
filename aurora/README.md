# Gerador de Rotas ITX — Flask (com modelos)

## Como rodar
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/macOS: source .venv/bin/activate
pip install -r requirements.txt
python app.py

Abra http://127.0.0.1:5000

## Modelos
Baixe modelos pela interface (links no formulário) ou diretamente:
- /modelo/xlsx -> MODELO_PLANILHA_ROTAS.xlsx
- /modelo/csv  -> MODELO_PLANILHA_ROTAS.csv
